# Dotfiles

## Zsh

Zsh setup is divided into 4 files

  * .zshrc loads all files from .zsh directory
  * aliases.zsh holds all my aliases
  * env.zsh holds all environment variables
  * keys.zsh holds all keybindings for terminal

## Nvim 

  * init.vim loads all plugins from dein package manager and sources all files from sources directory
  * configs.vim holds all configurations for neovim and colorscheme setup
  * keybindings.vim holds all keybindings for neovim and plugins
  * plug_configs.vim holds all configurations for all the plugins used

## Git
  
  * .gitconfig as all my git configurations (colors, aliases, etc...)
